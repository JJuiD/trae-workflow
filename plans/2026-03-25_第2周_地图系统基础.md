# 第2周：地图系统基础

## 任务目标

实现俯视角游戏地图，支持中央水晶和四个玩家方位（上下左右）。

## 详细任务拆解

### 2.1 地图数据结构
- [x] 定义地图网格系统（默认 25x25 或可配置）
- [x] 坐标系统：笛卡尔坐标系，原点为地图中心
- [x] 地块类型：空地、障碍物、出生点

### 2.2 中央水晶
- [x] 水晶位置固定在地图中心
- [x] 水晶血量属性
- [x] 水晶受击时的视觉反馈

### 2.3 玩家方位
- [x] 定义四个玩家方位：上、下、左、右
- [x] 每个方位的玩家出生点
- [x] 方位与水晶的相对位置关系
- [x] **玩家位置**：水晶相邻位置，紧凑布局便于快速支援
- [x] **独占规则**：4个方位互斥，先选先得

### 2.4 敌人出生
- [x] 敌人从屏幕外任意位置向水晶移动

### 2.5 地图渲染
- [x] Canvas 渲染地图网格
- [x] 渲染中央水晶
- [x] 渲染玩家方位标识
- [x] 简单的视觉风格（便于调试）

## 技术方案

### 地图坐标系统
```
       上(0, y_max)
         ↑
左(x_min, 0) → → → 中央(0, 0) → → → 右(x_max, 0)
         ↓
       下(0, y_min)
```

### 文件结构
```
src/
├── game/
│   ├── map.js           # 地图数据
│   ├── crystal.js       # 水晶
│   └── player_position.js
└── renderer/
    └── map_renderer.js
```

### 玩家方位坐标
```
       上(12, 11)
         ↑
左(11, 12) → 中央(12, 12) → 右(13, 12)
         ↓
       下(12, 13)
```
