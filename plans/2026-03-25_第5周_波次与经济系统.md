# 第5周：波次系统与经济系统

## 任务概述

实现游戏波次生成系统和金币经济系统，包括敌人波次管理、金币获取、建造消耗等核心机制。

## 任务目标

1. 波次生成系统（WaveSystem）
2. 金币经济系统
3. 敌人类型扩展
4. 难度曲线设计

## 详细任务拆解

### 1. 波次生成系统

- [ ] WaveManager - 波次管理
- [ ] WaveConfig - 波次配置（每波敌人数、间隔、类型）
- [ ] 波次难度递增算法
- [ ] Boss波次设计

### 2. 经济系统

- [ ] 金币获取（击杀敌人、过关奖励）
- [ ] 建造成本
- [ ] 升级消耗
- [ ] 技能抽取消耗

### 3. 敌人类型扩展

- [ ] 快速小怪（低血量、高移速）
- [ ] 装甲怪（高防御）
- [ ] 精英怪
- [ ] Boss敌人

## 技术方案

### 波次配置结构

```javascript
const WAVE_CONFIG = {
    1: { enemies: 5, interval: 3, types: ['basic'] },
    2: { enemies: 8, interval: 2.5, types: ['basic', 'fast'] },
    3: { enemies: 10, interval: 2, types: ['basic', 'fast', 'armor'] },
    // ...
};
```

### 经济配置

| 项目 | 数值 |
|------|------|
| 初始金币 | 100 |
| 击杀奖励 | 10 |
| 过关奖励 | 50 × 波次 |
| 建塔费用 | 50 |
| 技能抽取 | 30 |
