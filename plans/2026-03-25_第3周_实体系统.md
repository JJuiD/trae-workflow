# 第3周：实体系统

## 任务目标

设计并实现游戏实体系统，包括 Entity 基类、防御塔和敌人的抽象基类，以及基础属性管理。

## 详细任务拆解

### 3.1 Entity 基类
- [x] 实体唯一标识（ID）
- [x] 实体类型（type: 'tower' | 'enemy' | 'crystal'）
- [x] 位置属性（x, y 像素坐标）
- [x] 碰撞半径（collisionRadius）
- [x] 是否可碰撞（hasCollision）
- [x] 基础属性：
  - 生命值（hp, maxHp）
  - 魔法值（mp, maxMp）
  - 防御力（defense）：减伤比例
  - 攻击力（damage）
  - 暴击率（critRate）：暴击概率 0-1
  - 暴击伤害（critDamage）：暴击时的伤害倍率
  - 攻击速度（attackSpeed）：攻击间隔倍率，实际间隔 = attackCooldown / attackSpeed
  - 攻击间隔（attackCooldown）：基础攻击间隔
  - 当前冷却（currentCooldown）：当前剩余冷却时间
- [x] 存活状态（isAlive）
- [x] update(deltaTime) 方法
- [x] render(ctx) 方法
- [x] takeDamage(amount) 方法
- [x] 碰撞检测函数（checkCollision, resolveCollision）

### 3.2 防御塔基类 (Tower)
- [x] 继承自 Entity
- [x] 攻击范围（range）
- [x] 目标敌人引用
- [x] findTarget(enemies) 方法：选择最近敌人
- [x] attack(target) 方法：攻击逻辑（包含暴击判定）
- [x] 被动技能引用
- [x] 攻击指示线
- [x] 血条显示
- [x] 碰撞半径（collisionRadius = 15）

### 3.3 敌人基类 (Enemy)
- [x] 继承自 Entity
- [x] 移动速度（speed）
- [x] 初始目标：水晶
- [x] **攻击逻辑**：
  - 优先朝水晶移动
  - 进入**塔的攻击范围**时，切换目标为塔
  - 摧毁塔后继续朝水晶移动
- [x] 攻击水晶伤害（damageToCrystal）
- [x] 死亡判定与移除
- [x] 碰撞半径（collisionRadius = 3）
- [x] 渲染尺寸缩小（8px）

### 3.4 实体管理器 (EntityManager)
- [x] 管理所有实体列表
- [x] addEntity(entity)
- [x] removeEntity(entityId)
- [x] getEntitiesByType(type)
- [x] updateAll(deltaTime)
- [x] renderAll(ctx)
- [x] cleanup() 清理死亡实体
- [x] 敌人间碰撞检测与响应

- [x] Entity 基类可以被正确继承
- [x] Tower 可以寻找最近敌人并发起攻击
- [x] Enemy 可以向水晶移动
- [x] EntityManager 可以管理多个实体并统一更新/渲染
- [x] 实体系统与现有 Game 类集成

- [x] 塔和敌人有独立的碰撞半径

## 技术方案

### 实体类结构
```
Entity (基类)
├── id: string
├── type: string
├── x, y: number
├── collisionRadius: number
├── hasCollision: boolean
├── hp, maxHp: number
├── mp, maxMp: number
├── defense: number (减伤比例)
├── damage: number
├── critRate: number (暴击率 0-1)
├── critDamage: number (暴击伤害倍率)
├── attackSpeed: number (攻击速度倍率)
├── attackCooldown: number (基础攻击间隔)
├── currentCooldown: number (当前冷却时间)
├── isAlive: boolean
├── update(deltaTime)
├── render(ctx)
└── takeDamage(amount)

Tower (继承 Entity)
├── range: number
├── collisionRadius: number
├── passiveSkill: Skill
├── findTarget(enemies)
├── attack(target) - 含暴击判定
└── update(deltaTime)

Enemy (继承 Entity)
├── speed: number
├── damageToCrystal: number
├── collisionRadius: number
├── findTargetTower(towers)
├── attackTower(tower)
└── attackCrystal(crystal)

碰撞系统
├── checkCollision(entityA, entityB)
└── resolveCollision(entityA, entityB)
```
