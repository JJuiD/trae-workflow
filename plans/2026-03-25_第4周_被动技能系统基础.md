## 技术方案

### 1. 技能数据结构

每个技能属于特定等级，升级就是替换成另一个等级的技能：

```javascript
class PassiveSkill {
  constructor(config) {
    this.id = config.id;           // 技能唯一ID
    this.name = config.name;       // 技能名称
    this.description = config.description; // 技能描述
    this.level = config.level;     // 技能等级（1-5）
    this.effect = config.effect;   // 技能效果
  }
}
```

### 3. 技能升级机制

**核心规则**：升级不是强化当前技能，而是从更高等级的技能池中随机获得新技能。

```javascript
class SkillSlot {
  constructor(index) {
    this.index = index;    // 槽位编号 0-8
    this.skill = null;     // 当前技能
    this.level = 0;        // 当前等级
  }

  // 升级：从 level+1 技能池中随机抽取新技能
  upgrade(availableSkills) {
    if (this.level >= 5) return false;
    const nextLevel = this.level + 1;
    const pool = availableSkills.filter(s => s.level === nextLevel);
    if (pool.length === 0) return false;
    this.skill = pool[Math.floor(Math.random() * pool.length)];
    this.level = nextLevel;
    return true;
  }

  canUpgrade() {
    return this.level < 5;
  }
}
```
