## 技术方案

### 1. 技能数据结构

每个技能属于特定等级，升级就是替换成另一个等级的技能：

```javascript
class PassiveSkill {
  constructor(config) {
    this.id = config.id;           // 技能唯一ID
    this.name = config.name;       // 技能名称
    this.description = config.description; // 技能描述
    this.level = config.level;     // 技能等级（1-5）
    this.effect = config.effect;   // 技能效果
  }
}
```

### 2. 技能效果类型

| 类型 | 说明 | 示例 |
|------|------|------|
| damage_boost | 伤害提升 | +10% 攻击力 |
| attack_speed | 攻速提升 | +5% 攻速 |
| range_boost | 范围提升 | +8% 攻击范围 |
| heal | 生命恢复 | 每秒 +2 HP |
| shield | 护盾 | 受到伤害时生成护盾 |

### 2.1 主动技能 vs 被动技能

| 类别 | 说明 | 触发方式 |
|------|------|----------|
| 被动技能 | 常驻加成，无需触发 | 始终生效 |
| 主动技能 | 需要主动释放 | 玩家手动触发 |
| 定时技能 | 周期性触发 | 每隔N秒自动触发 |

```javascript
class PassiveSkill {
  constructor(config) {
    this.id = config.id;
    this.name = config.name;
    this.description = config.description;
    this.level = config.level;
    this.category = config.category; // 'passive' | 'active' | 'timed'
    this.effect = config.effect;
  }
}
```

### 2.2 技能效果实现机制

```javascript
// 被动加成类
const PASSIVE_EFFECTS = {
  damage_boost: (entity, value) => {
    entity.attackDamage *= (1 + value);
  },
  attack_speed: (entity, value) => {
    entity.attackSpeed *= (1 + value);
  },
  range_boost: (entity, value) => {
    entity.attackRange *= (1 + value);
  },
};

// 定时伤害类
const TIMED_EFFECTS = {
  fireball: (game, entity, config) => {
    // 每隔N秒发射火球
    setInterval(() => {
      const target = game.entityManager.getNearestEnemy(entity.x, entity.y);
      if (target) {
        game.projectiles.spawn({
          x: entity.x,
          y: entity.y,
          targetId: target.id,
          damage: config.damage,
          type: 'fireball'
        });
      }
    }, config.interval * 1000);
  },
  lightning_strike: (game, entity, config) => {
    setInterval(() => {
      const targets = game.entityManager.getEnemiesInRange(
        entity.x, entity.y, config.range
      );
      targets.forEach(target => {
        target.takeDamage(config.damage);
      });
    }, config.interval * 1000);
  },
};
```

### 3. 技能升级机制

**核心规则**：升级不是强化当前技能，而是从更高等级的技能池中随机获得新技能。

```javascript
class SkillSlot {
  constructor(index) {
    this.index = index;    // 槽位编号 0-8
    this.skill = null;     // 当前技能
    this.level = 0;        // 当前等级
  }

  // 升级：从 level+1 技能池中随机抽取新技能
  upgrade(availableSkills) {
    if (this.level >= 5) return false;
    const nextLevel = this.level + 1;
    const pool = availableSkills.filter(s => s.level === nextLevel);
    if (pool.length === 0) return false;
    this.skill = pool[Math.floor(Math.random() * pool.length)];
    this.level = nextLevel;
    return true;
  }

  canUpgrade() {
    return this.level < 5;
  }
}
```
